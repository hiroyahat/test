git、github研修

git
分散型バージョン管理システム
ファイルの変更履歴を残せる

github
gitを利用したWebサービスの名称
ソースコードを保存・公開が可能
プルリクエスト（レビュー依頼）を行うことができる

メリット
最新のデータをわかりやすく管理できる
誰が、いつ、どこを変更したの履歴が残る
複数人で同時に開発ができる
プルリクエスト機能によりコードレビューができる

概念
サーバー上のファイルをローカルにコピー
ローカルで編集した内容をサーバー側にUPを繰り返し開発を行う
サーバーのリポジトリからクローン
ファイルシステムで編集してコミット

SSH
IPアドレスとパスワードで入力
SSHキー公開鍵認証
パスワードまでも入力する必要がない

秘密鍵（idrsa）と公開鍵（id_rsa.pub）

リポジトリ：ファイルの保存場所のこと。ローカルとリモートが存在
クローン：リモートリポジトリをローカル環境として構築すること
プル：リモートリポジトリから最新のデータを取得
コミット：ローカルリポジトリに変更を保存
